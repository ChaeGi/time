시간계산기 최종판

이 파일은 인터넷 연결 없이 사용할 수 있도록 만든 개인용 오프라인 시간계산기입니다.

주요 기능
- 여러 시간 합산
- 시작 시각과 종료 시각의 시간 차이 계산
- 기준 시각에 시간을 더하거나 빼는 시각 계산
- 계산 기록 저장
- 기록 선택 삭제 및 전체 삭제
- 합산 목록 활성/비활성 토글
- 시간/분 분리 편집
- 분 입력 0~59 제한
- 입력패드 고정 크기 유지

사용 방법
1. index.html 파일을 브라우저에서 엽니다.
2. iPad 홈 화면 앱처럼 사용하려면 HTTPS 서버에 업로드한 뒤 Safari에서 홈 화면에 추가합니다.
3. PWA 설치 후에는 캐시된 파일로 오프라인 실행됩니다.
